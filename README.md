# 加密说明书
![加密说明书](public/LowPower.png)
加密货币实操手册 - 从入门到精通的完整指南。基于 Astro 构建，轻量快速。

![加密说明书](public/LowPower3.png)
![加密说明书](public/LowPower2.png)

## Features

- 简洁的单色设计
- 轻量快速加载
- 内容集合管理博客文章和页面
- 标签系统组织内容
- 响应式网格布局
- 高效的图片处理
- 使用 astro-imagetools

## Getting Started

```bash
npm install
npm run dev
npm run build  
npm run preview
```

## Content Structure

- `/src/content/blog/` - 博客文章
- `/src/content/page/` - 静态页面
- `/public/images/` - 图片资源
